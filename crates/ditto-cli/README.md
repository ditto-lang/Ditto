```
ditto
putting the fun in functional

USAGE:
    ditto <SUBCOMMAND>

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information

SUBCOMMANDS:
    bootstrap    Bootstrap a new project
    make         Build a project
    fmt          Format ditto code
    lsp          Start up the language server
```
